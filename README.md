# ⚙️ 前端基础框架设计说明

## 🧩 技术基础
- 使用 **Bun** 作为运行时与构建环境，提供高性能与快速启动体验。  
- 基于 **Rsbuild** 构建，支持极速编译与灵活插件扩展。

## 🌍 国际化（i18n）
- 采用 **按需加载** 策略，仅在首次访问或切换语言时加载对应语言包。  
- 使用 `vue-i18n` 管理多语言，支持运行时动态注册与缓存预加载。

## 🧭 路由管理
- 路由采用 **模块化自动注册**，每个功能模块独立维护自身路由文件。  
- 系统自动为语义路径（如 `/home`、`/chat`）分配 **数字别名**（如 `/1`、`/2`）。  
- 路由路径经过 **数字化加密处理**，隐藏真实结构，对外仅暴露数字路径。  
- 运行时通过映射表解析数字路径，实现可逆的安全访问。

## 🧱 状态管理
- 基于 **Pinia** 的模块化状态管理，每个业务模块独立注册。  
- 支持按需加载与销毁，避免全局状态冗余。

## 📦 公共依赖与组件
- 公共依赖（Vue、Vant、Axios、Vue Router、Vue I18n 等）通过 **CDN 外部引入**，减少打包体积。  
- 公共组件集中维护，统一版本管理，支持自动导入与按需加载。
